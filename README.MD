#WEBRTC & MSE demo

#usage
- git clone
- npm install static -g & npm install browserify -g
- npm install
- npm run build
- node server.js
- static -p 8081
- open http://localhost:8081/media_source.html
- open http://localhost:8081/index.html#1 in chrome
- open http://localhost:8081/index.html in firefox
